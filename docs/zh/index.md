# kuku 文档

- [How It Works](how-it-works/index.md) - 理解运行模型、Session 流程、Memory 和权限。
- [Reference](reference/index.md) - 查命令、配置、环境变量、格式、API 和术语。
- [Architecture](architecture/index.md) - 查看 crate 边界、host apps、Prompt 装配和内部设计。
